# Archive Layout

`archive/` 用于存放不直接参与运行、但需要长期保留的毕业设计资料和学校过程材料。

## 目录说明

- `graduation-materials/templates/`
  - 论文模板、参考模板、格式样例。

- `graduation-materials/deliverables/`
  - 个人阶段性产出，例如中期报告、外文翻译、论文 Word/PDF 等。

- `graduation-materials/notes/`
  - 毕业设计过程记录、工作笔记。

- `graduation-materials/visuals/`
  - 答辩图、展示图、截图等视觉材料。

- `school-docs/`
  - 学校或学院要求材料，例如任务书、开题报告、审批表、检查表。

## 原则

- 模板和正式提交物分开存放。
- 学校提供的过程材料统一归到 `school-docs/`。
- 运行代码不要直接依赖 `archive/` 中除模板外的资料。
