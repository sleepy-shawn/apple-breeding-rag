# Workspace Layout

`workspace/` is the automation-friendly data area for this project.

- `default/source/`: raw downloads and external fetch outputs
- `default/library/papers/`: curated per-paper folders with `raw/` and `processed/`
- `default/manifests/`: checklist, ingest manifest, and pipeline control tables
- `default/evaluation/`: benchmark questions, answers, and scoring artifacts
- `default/reports/`: coverage and pipeline reports
- `default/state/`: incremental pipeline checkpoints
